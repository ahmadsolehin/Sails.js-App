# ArticleBase

a [Sails](http://sailsjs.org) CRUD application to manage articles

### Version
1.0.0

## Usage


### Installation

Install the dependencies

```sh
$ npm install sails -g
```

```sh
$ npm install
```

### Serve
To serve in the browser

```sh
$ sails lift
```
